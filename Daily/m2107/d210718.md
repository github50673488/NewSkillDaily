NUXT JS 再学习
=

### 目录

+ .nuxt  
  --- 不用动
+ assets  
  ----放logo等静态资源
+ components 组件
+ layouts  
  --- 布局，default.vue
+ middleware  
  --- 比如，用户认证，log等
+ pages 页面
+ plugins  
  nuxt官方非模块（nuxt支持很多常用库如axios，称为模块）的引用放在这里
+ static  
  ---也是静态资源，不归vue管了，比如说 放 landing page （着陆页面。(广告效果)），
+ store  
  状态管理，数据交换

### 文件

+ jsconfig.json？   
  目录中存在jsconfig.json文件表示该目录是JavaScript项目的根目录。 jsconfig.json文件指定根文件和JavaScript语言服务提供的功能选项
+ nuxt.config.js   
  --核心配置文件

+ package.json  
  这个不用多说了，npm包管理
